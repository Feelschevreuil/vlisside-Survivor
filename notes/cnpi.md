# Visuel

## Aspect général
- Épuré!
- Dynamique
- Ergonomique
- Visible sur portable (réactif/bootstrap) 
- Couleurs indéterminer 
  - Pas de rose, mauve
  - Pas flash, ne fatigue pas les yeux
  - Passe les années
